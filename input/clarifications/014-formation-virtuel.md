# Cas des formations dites « virtuelles »

Une formation est dite virtuelle (ou « CV »), également appelée « classe virtuelle », lorsqu’elle est dispensée via un logiciel de visioconférence.

Pour ces formations, le lieu doit être :

"2 allée du Commandant Charcot 77200 TORCY (France)".

## Règle d’identification

Une formation est virtuelle lorsque son code étendu se termine par :

- `CV1`, ou
- `CV<n>` où `<n>` est un chiffre (`0`…`9`).

Cette règle doit être implémentée pendant l’analyse des formations.

## Paramètres

Ajouter dans la page react des paramètres un bouton :

"corriger adresse des formation virtuel"

Objectif : éviter de relancer une analyse coûteuse uniquement pour corriger le cas des formations virtuelles.
