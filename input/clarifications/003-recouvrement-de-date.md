# Recouvrement de dates

Deux formations ne peuvent pas avoir lieu aux mêmes dates. Si cela arrive, cela signifie qu’il y a une erreur dans le traitement des données issues des emails.

Il faut le signaler dans l’interface web.

## Proposition

Ajouter une section **ERREURS** dans la page des paramètres.

Cette section liste les formations problématiques (par exemple : recouvrement de dates avec une autre formation).
