# Mise à jour des coordonnées GPS

On constate que, dans de nombreux cas :

- les coordonnées GPS ne sont pas trouvées par le service de géolocalisation ;
- les coordonnées GPS renvoyées ne sont pas du tout exactes.

Il faut un système permettant de mettre à jour facilement les coordonnées GPS, via une carte où l’on peut indiquer d’un clic l’endroit exact où la formation a eu lieu.

Plus precisement, le user flow doit etre a peu pres cela :

1. L'utilisateur selectionne une formation. (dans la page Formations)
2. Il est redirige sur la page qui donne les details de la formation.
3. Sur cette page on voit une carte (avec la position GPS si elle existe).
4. L'utilisateur peut si il le veut mettre a jour cette position GPS en cliquant sur un bouton "corriger la position" puis cliquer a l'endroit de la nouvelle position, puis en cliquant sur un bouton "valider la nouvelle position"
