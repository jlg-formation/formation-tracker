# Codes de formation ORSYS

Chez ORSYS, chaque sujet de formation possède un code.

## Exemples

- Formation AngularJS → code `FAN`
- Formation Ergonomie web → code `IHM`
- Formation Design web sur 5 jours → code `DHM`
- etc.

## Extension du code

Le code peut ensuite être étendu avec une lettre et un chiffre.

- La lettre indique le lieu de formation : les centres ORSYS ont chacun leur identifiant.
  - Exemple : `P` pour Paris, `E` pour Lille, `N` pour Nantes, etc.
- Il y a eu une réforme : ces lettres sont passées à un format à deux lettres.
  - Exemple : `PA` pour Paris, `NA` pour Nantes, `AX` pour Aix-en-Provence, etc.

## Cas particulier : `(FR)`

Dans certains emails, on trouve `(FR)` : ce n’est pas un centre, mais la langue dans laquelle la formation est donnée.
